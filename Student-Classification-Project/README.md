# Student Classification Model

## Project Description

This project builds a basic machine learning classification model using Python and a custom student dataset. The model predicts whether a student will Pass or Fail based on:

* Hours Studied
* Attendance Percentage
* Completed Assignments

## Features

* Loads and analyzes dataset
* Splits data into training and testing sets
* Uses Decision Tree Classification
* Trains a prediction model
* Evaluates model accuracy
* Accepts user input for prediction

## Technologies Used

* Python
* Pandas
* Scikit-learn

## Project Structure

## How to Run

Install dependencies:

pip install pandas scikit-learn

Run:

python classification_model.py
