# Simple Rule-Based Chatbot

## Project Description
This project is a simple rule-based chatbot developed using Python. The chatbot responds to predefined user inputs using decision-making logic (`if-elif-else`) and runs continuously until the user chooses to exit.

## Features
- Handles greeting messages like:
  - Hi
  - Hello
  - Hey
- Responds to common questions
- Supports exit commands
- Uses `if-elif-else` statements
- Runs in a continuous loop
- Demonstrates basic AI chatbot concepts

## Technologies Used
